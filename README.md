# Aarware Core Tools

A comprehensive Unity package for quick game bootstrapping with multi-platform service support.

## Features

- **Multi-Platform Services**: Account, Data Storage, Stats, Achievements, Leader Boards
- **Localization System**: Dynamic language support with automatic translation components
- **Networking**: Support for Photon, custom servers, and Steam
- **Utilities**: Singleton patterns, remote image loading, avatar generation
- **Unified Editor Window**: All-in-one tabbed interface for managing everything
  - Session data (accounts, stats, achievements, saves)
  - Cache management
  - Localization tools
  - Leaderboard management
  - Service configuration
  - Scene management with enum generation

## Quick Start

1. Open Unity
2. Go to `Aarware > Core Tools Window`
3. Use the tabs to manage different aspects of your game

## Installation

Add this package to your Unity project via the Package Manager using the git URL or by placing it in your Packages folder.

## Requirements

- Unity 2020.3 or later
- TextMeshPro (automatically installed as dependency)
